## コンテナのビルドと起動
全部WSL上で
 - clone
    `git clone https://github.com/waneen/rust_100.git`
 - 移動
    `cd rust_100`
 - コンテナのビルドと起動
    `docker-compose up -d --build`
 - プロジェクトを開く
    `code .`
 - コンテナに入って作業
 　左下の青枠で`WSL: Ubuntu`等となっているところをクリック、`Reopen in Container`で完了
